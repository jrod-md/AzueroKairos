# First Delivery

This first delivery establishes the official clean repository scaffold for Azuero Kairós.

## Included

- Project README and scientific/compliance documentation.
- Minimal Python package under `src/azuero_kairos/`.
- Configuration placeholders under `configs/`.
- Empty output folders tracked with `.gitkeep`.
- Minimal dashboard scaffold under `dashboard/`.

## Not Included

- Copernicus API acquisition code.
- Generated Sentinel results.
- Processed CSV outputs.
- Generated Confidence Briefs.
- Cloud infrastructure.
- AWS code.
- Twilio or messaging integrations.

## Acceptance Criteria

- The repository can be installed from `requirements.txt`.
- Official outputs are not pre-populated.
- Scientific limits are explicit and visible.
- The clean-build policy is documented.
- Future scripts have clear input and output locations.
